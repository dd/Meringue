Добро пожаловать, Путник! Это документация Meringue.
====================================================

Meringue это пакет с набором готового функицонала для `Django <http://www.djangoproject.com>`_,
она включает в себя такие вещи как миксины для моделей, абстрактные модели, утилита для работы с изображениями,
набор декораторов, набор кастомных полей и прочее, подробне можно ознакомиться в этой документации
